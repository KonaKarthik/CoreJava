#Tution fee Calculation

Student's grade level (1-12)
Base tution fee
Discount percentage
Academic Topper status (e.g., whether the student is an acedamic topper or not )

Determine the discount based on the following rules:

    -> For the students in grades 9-12
        -> If they are academic toppers, apply a 20% discount.
        -> Otherwise, apply a 10% discount.
    -> For the students in grades 6-8
        -> apply a 5% discount
    -> Below grade 6
        -> No discount
    
    -> Apply Additional Discount
        -> For grade 10, add an additional 3% discount.
        -> For grade 12, add an additional 5% discount.
        -> No additional discounts for the other grades.
    
    -> Calculate Actual fee
        -> Base fee - Discount.

    -> Display Output
        -> Student grade.
        -> Student Base fee.
        -> Discount Amount.
        -> Fee to pay.
